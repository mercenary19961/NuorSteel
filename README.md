# Nuor Steel — Corporate Website

The official website of **Nuor Steel Industry Company** (شركة نور للصناعات الحديدية), with a built-in admin panel for managing products, certificates, careers, and content.

---

## About the Site

Nuor Steel's website is a bilingual (English / Arabic) corporate platform featuring:

- A public-facing site showcasing the company, products, certifications, quality standards, and careers
- A built-in content management system (admin panel) for updating content without a developer
- Automatic handling of contact forms, career applications, and newsletter signups
- Fully responsive design for desktop, tablet, and mobile

---

## Public Website

Visitors can browse:

| Page | Purpose |
|------|---------|
| **Home** | Overview of the company, products, values, and news |
| **About Us** | Company story, vision, mission, and journey |
| **Products** | Detailed product catalog (TMT Bars, Billets) with specifications |
| **Quality** | Quality assurance standards and processes |
| **Certificates** | ESG, Quality, and Governance certificates with PDF viewer |
| **Career** | Current job openings and application forms |
| **Contact** | Contact form for inquiries, partnerships, and quotations |

The website automatically switches between English and Arabic based on the visitor's choice, with a language toggle in the header.

## Admin Panel

The admin panel lets authorized team members update the website without writing code.

### Access

- **URL:** `https://[your-live-domain]/admin/login`
- **Login:** Provided separately (each team member has their own account)

### What You Can Manage

- Products and product specifications
- Certificates (upload PDFs, set categories and expiry dates)
- Job listings and applications
- Site content (text blocks across all public pages)
- Media library (images and PDFs)
- Contact form submissions
- Newsletter subscribers
- LinkedIn posts featured on the homepage
- Site settings (company contact info, email recipients)
- User accounts (admin only)

## Safety Features

The admin panel is designed so nothing is lost accidentally:

- **30-second Undo button** after every change
- **Permanent Change Log** with full history and revert support (admin only)
- **Self-protection rules** that prevent locking yourself out of the system
- **Type-to-confirm prompts** on all destructive actions
